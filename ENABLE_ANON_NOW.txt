┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: Open Supabase Dashboard                                             │
│                                                                              │
│ 1. Open your browser                                                         │
│ 2. Go to: https://supabase.com/dashboard                                    │
│ 3. Log in if needed                                                          │
│ 4. Click on your "CareNest" project                                          │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: Navigate to Authentication                                          │
│                                                                              │
│ 1. Look at the LEFT SIDEBAR                                                  │
│ 2. Find and click "Authentication" (🔐 icon)                                 │
│ 3. You'll see several tabs at the top                                        │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: Go to Providers Tab                                                 │
│                                                                              │
│ 1. Click the "Providers" tab (at the top)                                    │
│ 2. You'll see a list of authentication providers                             │
│    (Email, Google, GitHub, etc.)                                             │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 4: Find Anonymous Sign-ins                                             │
│                                                                              │
│ 1. SCROLL DOWN on the Providers page                                         │
│ 2. Look for a section called "Anonymous Sign-ins"                            │
│    (It's usually near the bottom)                                            │
│ 3. You'll see a toggle switch that's currently OFF (gray)                    │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 5: Enable Anonymous Sign-ins                                           │
│                                                                              │
│ 1. Click the toggle switch to turn it ON                                     │
│    (It should turn green/blue)                                               │
│ 2. A "Save" button should appear                                             │
│ 3. Click the "Save" button                                                   │
│ 4. Wait for confirmation message                                             │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 6: Verify It's Enabled                                                 │
│                                                                              │
│ 1. The toggle should now be ON (green/blue)                                  │
│ 2. You should see "Anonymous sign-ins enabled" or similar message            │
└──────────────────────────────────────────────────────────────────────────────┘
